# Fotos dos Closers

Coloque aqui a foto de cada closer, em PNG, nomeada com o **primeiro nome em minúsculo**:

- `leonardo.png`
- `gustavo.png`
- `thiago.png`

A foto aparece automaticamente nos cards e no ranking de closers. Se o arquivo
não existir, o dashboard mostra as iniciais do closer como fallback.

Dica: use imagens quadradas (ex.: 400x400) — elas são recortadas em círculo.
Depois de adicionar/atualizar uma foto, faça o commit/push para o site atualizar.
